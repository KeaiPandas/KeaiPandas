## ⭐ 项目

📝 [Minsight-Deploy](https://github.com/KeaiPandas/Minsight-Deploy) - 面向会议场景的 AI 结构化工作台。把会议转写文本自动转成可读纪要、参会人、关键议题、待办事项、决策结论与证据片段，支持派生任务、跨会议提示和飞书集成。采用多 Agent 工作流（规范化 → 议题抽取 → 待办/决策识别 → 过滤校验与修复回环），并有独立的 Lab Benchmark 做回归评测

🏠 [realty-agent](https://github.com/KeaiPandas/realty-agent) - 在 Windows 本地运行的房产销售 AI 工具。读取本地聊天数据库，用大模型从聊天记录中提炼客户画像（需求 / 预算 / 意向 / 跟进策略），自动同步到飞书或钉钉多维表格，并提供线索情报看板（风险预警、每日简报）与半自动 / 全自动回复 Bot。FastAPI + LangGraph + Vue 3

🎮 [tabquest](https://github.com/KeaiPandas/tabquest) - 把标签页变成游戏。100% 本地运行的 Chrome 插件，关标签页、完成待办都能拿经验升级

🤖 [agile-ai-team](https://github.com/KeaiPandas/agile-ai-team) - 基于 OpenClaw 的多 Agent 协作系统，用 3 个 Agent（PO / Dev / Quality）模拟完整 Scrum 团队，覆盖需求拆解、Sprint 规划、开发、测试、发布全链路。每个 Agent 拥有独立人格、职责边界、记忆系统与定制 Skills，通过 Agent 间通信实现端到端交付

🔍 [web-search-mcp](https://github.com/KeaiPandas/web-search-mcp) - 一个 MCP server，给任意 coding agent 提供网页搜索能力
