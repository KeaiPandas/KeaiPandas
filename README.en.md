## ⭐ Projects

📝 [Minsight-Deploy](https://github.com/KeaiPandas/Minsight-Deploy) - An AI workbench for meetings. Turns raw transcripts into readable minutes, attendees, key topics, action items, decisions, and evidence snippets — with derived tasks, cross-meeting hints, and Feishu integration. Built as a multi-agent workflow (normalize → extract topics → identify todos/decisions → filter, verify, repair), with a separate Lab Benchmark for regression evaluation

🏠 [realty-agent](https://github.com/KeaiPandas/realty-agent) - A Windows-local AI assistant for property sales. Reads your local chat database, distills customer profiles from conversations (needs, budget, intent, follow-up strategy), syncs them to Feishu or DingTalk sheets, and adds a lead intelligence dashboard plus a semi/fully automatic reply bot. FastAPI + LangGraph + Vue 3

🎮 [tabquest](https://github.com/KeaiPandas/tabquest) - Gamify your tabs. A 100% local Chrome extension where closing tabs and finishing to-dos earns XP and levels you up

🤖 [agile-ai-team](https://github.com/KeaiPandas/agile-ai-team) - A multi-agent system on OpenClaw where three agents (PO / Dev / Quality) run a full Scrum team — backlog, sprint planning, development, testing, release. Each agent has its own personality, responsibility boundary, memory, and custom skills, and they hand work to each other end to end

🔍 [web-search-mcp](https://github.com/KeaiPandas/web-search-mcp) - An MCP server that gives any coding agent web search

## 🧭 How I build

- **Local first.** If it can run on your own machine, it should.
- **One job, done well.** A tool that does one thing you'd miss beats a platform you forget.
- **Ship, then listen.** Every project here started as something I needed myself.
